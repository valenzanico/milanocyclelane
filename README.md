## Getting started

- clone the git repo
  ```
  git clone https://github.com/valenzanico/milanocyclelane.git
  ```
- install npm packages
  ```
  npm install
  ```
- run the project
  ```
  npm run frontend
  npm run backend
  ```
> open the browser on `http://localhost:5000`
